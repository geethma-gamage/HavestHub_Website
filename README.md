Project Overview

HarvestHub Website is a React-based web application designed to:

- Display agricultural products with details
- Showcase gallery images
- Provide contact information and forms
- Allow user login functionality
- Support easy navigation and modern UI design

- Features

- Responsive homepage with **About, Gallery, and Contact sections**
- **Login** functionality for users/admin
- **Gallery** with product images
- **Contact form** for inquiries
- Modern UI with CSS styling
